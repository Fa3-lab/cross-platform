/* Modern CSS Grid */ .modern-grid { display: grid; grid-template-columns: repeat(3, 1fr); grid-template-rows: 100px 200px 100px; gap: 20px; } .header-item { grid-column: 1 / -1; /* Span all columns */ } .main-item { grid-column: 1 / 3; /* Span first two columns */ } .sidebar-item { grid-column: 3; /* Third column */ } .footer-item { grid-column: 1 / -1; /* Span all columns */ }

Modern Grid Features

  • grid-template-areas
  • grid-gap / gap property
  • fr units
  • repeat() function
  • minmax() function
  • auto-fit / auto-fill

IE 10-11 Grid

  • -ms-grid display
  • -ms-grid-columns
  • -ms-grid-rows
  • Manual positioning
  • No gap property
  • Limited functions

No Grid Support

  • IE 9 and below
  • Old Android browsers
  • Old iOS Safari
  • Opera Mini
  • UC Browser (old)

/* IE 10-11 Compatible Grid */ .ie-grid { display: -ms-grid; display: grid; /* IE syntax - must account for gaps manually */ -ms-grid-columns: 1fr 20px 1fr 20px 1fr; -ms-grid-rows: 100px 20px 200px 20px 100px; /* Modern syntax */ grid-template-columns: repeat(3, 1fr); grid-template-rows: 100px 200px 100px; gap: 20px; } .header-item { /* IE positioning */ -ms-grid-column: 1; -ms-grid-column-span: 5; /* Includes gap columns */ -ms-grid-row: 1; /* Modern syntax */ grid-column: 1 / -1; } .main-item { -ms-grid-column: 1; -ms-grid-column-span: 3; -ms-grid-row: 3; grid-column: 1 / 3; }
⚠️ IE Grid Limitations: IE's grid implementation is incomplete. It doesn't support grid-template-areas, auto-placement, or the gap property. You must manually position every grid item and account for gaps in your column/row definitions.

Progressive Enhancement Strategy

A complete solution that works across all browsers

/* Progressive Enhancement Approach */ /* 1. Base styles for all browsers */ .grid-container { margin: 20px 0; } .grid-item { margin-bottom: 20px; } /* 2. Float fallback for older browsers */ .grid-container::after { content: ""; display: table; clear: both; } .header, .footer { width: 100%; clear: both; } .main { width: 65%; float: left; margin-right: 5%; } .sidebar { width: 30%; float: right; } /* 3. Flexbox enhancement */ @supports (display: flexbox) { .grid-row { display: flex; margin-bottom: 20px; } .main { float: none; flex: 2; margin-right: 20px; } .sidebar { float: none; flex: 1; } } /* 4. CSS Grid enhancement */ @supports (display: grid) { .grid-container { display: grid; grid-template-columns: 2fr 1fr; grid-template-rows: auto 1fr auto; gap: 20px; grid-template-areas: "header header" "main sidebar" "footer footer"; } .grid-item { margin-bottom: 0; /* Reset margins */ } .main { margin-right: 0; grid-area: main; } .sidebar { grid-area: sidebar; } .header { grid-area: header; } .footer { grid-area: footer; } }
💡 Best Practices:
  • Use @supports queries to detect CSS Grid support
  • Start with the most basic layout (floats)
  • Layer on more advanced features progressively
  • Test thoroughly in target browsers
  • Consider using CSS Grid with autoprefixer for IE support
