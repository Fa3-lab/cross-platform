Responsive Images

Images that scale properly across all device sizes while maintaining aspect ratio.

/* Responsive Image CSS */ .responsive-image { width: 100%; height: auto; max-width: 800px; /* Optional max size */ } /* For high DPI displays */ @media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) { .responsive-image { image-rendering: -webkit-optimize-contrast; } }

Fluid Typography

This heading scales with viewport size

This paragraph text also adapts to different screen sizes using the clamp() function. The text remains readable across all devices while maintaining appropriate proportions.

On smaller screens, the text size decreases to fit more content, while on larger screens, it increases for better readability at a distance.

/* Fluid Typography CSS */ h2 { font-size: clamp(1.5rem, 4vw, 3rem); /* min: 1.5rem, preferred: 4vw, max: 3rem */ } p { font-size: clamp(0.9rem, 2.5vw, 1.1rem); line-height: 1.6; }

Responsive Navigation

Navigation that transforms from horizontal layout on desktop to hamburger menu on mobile.

/* Responsive Navigation CSS */ .nav { display: flex; justify-content: center; } .mobile-nav, .hamburger { display: none; } @media (max-width: 480px) { .nav { display: none; } .mobile-nav { display: block; } .hamburger { display: block; } .nav-menu { display: none; flex-direction: column; } .nav-menu.active { display: flex; } }

Common Breakpoints

Standard breakpoints used in responsive design:

Mobile Portrait

320px - 480px

Single column layout, hamburger menu, large touch targets

Mobile Landscape

481px - 640px

Two-column grid, compact navigation

Tablet Portrait

641px - 768px

Multi-column layout, medium spacing

Tablet Landscape

769px - 1024px

Desktop-like layout, full navigation

Desktop

1025px+

Full layout, maximum content width
